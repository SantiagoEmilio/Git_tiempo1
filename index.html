1 git clone Crea una copia del repositorio
2 git status Muestra el estado del repositorio
3 git add Añade archivos al área de preparación
4 git commit Guarda los cambios en el repositorio
5 git push Envía los cambios al repositorio remoto
6 git pull Actualiza el repositorio local con los cambios del remoto
7 git branch Lista, crea o elimina ramas
8 git checkout Cambia entre ramas o revisiones
9 git merge Une cambios de diferentes ramas
10 git log Muestra el historial de commits
